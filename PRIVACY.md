# プライバシーポリシー

**d-OP**（以下「本拡張機能」）は、ユーザーのプライバシーを尊重し、個人情報の保護に配慮しています。

## 収集する情報

本拡張機能は、以下の情報を**ユーザーのブラウザ内**にのみ保存します：

- プレイリスト（作品タイトル、エピソードタイトル、再生区間の開始・終了時間）
- 再生状態（現在再生中のプレイリストと位置）
- 設定（新規ウィンドウ/タブの開き方の選択）

これらのデータはすべて Chrome のローカルストレージ（`chrome.storage.local`）に保存され、**外部サーバーに送信されることは一切ありません**。

## 収集しない情報

本拡張機能は以下を**収集しません**：

- 個人を特定できる情報（氏名、メールアドレス、住所など）
- dアニメストアのログイン情報、認証トークン、クッキー
- 閲覧履歴、視聴履歴
- 位置情報
- その他あらゆる個人情報

## データの保存と削除

- すべてのデータはユーザーのブラウザ上にのみ存在します
- 拡張機能をアンインストールすると、保存されたデータは自動的に削除されます
- 拡張機能のオプションページから、プレイリストを手動で削除することもできます

## 第三者提供

本拡張機能は**いかなる第三者ともデータを共有しません**。データがブラウザ外に送信されることはありません。

## セキュリティ

本拡張機能は `chrome.storage.local` API を使用しており、データは Chrome ブラウザによって保護されています。拡張機能自体にはネットワーク通信を行うコードは含まれていません。
